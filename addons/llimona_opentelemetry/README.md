# Llimona Opentelemetry sensors

> [!CAUTION]
>  Work in progress

This submodule provides OpenTelemetry sensors for the Llimona framework.

Its goal is to help integrate OpenTelemetry into Llimona-based applications so developers can collect telemetry data, observe runtime behavior, and connect Llimona components with standard observability tooling.

The project is still under active development, so its API and behavior may change.
