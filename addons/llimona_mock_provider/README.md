# Llimona Mock provider

> [!CAUTION]
>  Work in progress

This submodule provides a mock provider for Llimona.

Its purpose is to help developers test features, validate integrations, and simulate provider behavior without depending on external services. It is intended for development, experimentation, and automated tests where predictable and controllable responses are useful.

The project is still under active development, so its API and behavior may change.
